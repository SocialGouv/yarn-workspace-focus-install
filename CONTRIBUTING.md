# Contributing

> Thanks for being willing to contribute !

## TL;DR

```sh
# Install
$ yarn

# Build
$ yarn build
# Watch the build
$ yarn build --watch

# Lint
$ yarn lint

# Type check
$ yarn typecheck --watch

# Test
$ yarn test
# Watch the test
$ yarn test --watch

# Build
$ yarn e2e
# Watch the e2e
$ yarn e2e --watch
```
